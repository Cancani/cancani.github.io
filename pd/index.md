# Polizei Leitfaden

## Funkcodes
- **10-1** Im Dienst
- **10-2** Ausser Dienst
- **10-3** Pause
- **10-4** Verstanden
- **10-20** Standort
- **Code 1** Normalfahrt
- **Code 3** Blaulicht + Sirene
- **Code 4** Einsatz beendet
- **Code 6** Verstärkung
- **11-99 / Code Red** Beschuss auf Beamte

## Grundregeln
- Immer Befehle befolgen
- Nur zu zweit auf Streife
- Schutzweste im PD ausziehen
- Dienstnummer angeben

## Rechte vorlesen
- Recht zu schweigen
- Alles kann Gericht verwendet werden
- Recht auf Anwalt
- Verständnis prüfen

## O-Tablet
Zentrale Datenbank für Personen, Fahrzeuge und Akten

## Ausrüstung
- Westen, Handschellen, Fussfesseln
- UV-Lampe, GSR-Kit
- Essen/Trinken

## Waffen (Aspirant)
- Taser
- Schlagstock
- Taschenlampe

## Sperrzonen
Kann ausgerufen werden bei Gefahr, Raub, Schusswechsel

## Festnahme Ablauf
1. Fesseln  
2. Durchsuchen  
3. Rechte vorlesen  
4. Fotos machen  
5. Busse / Haft
